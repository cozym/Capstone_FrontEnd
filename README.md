# Capstone_FrontEnd

**git 초기 등록**

1. 원하는 로컬 저장소 경로에서 git init

2. git add .

3. git commit -m "커밋메세지"

4. git checkout -b "브랜치명"

5. git remote add origin/lim <원격저장소 링크>

6. git push -u origin/<각자 브랜치명>
